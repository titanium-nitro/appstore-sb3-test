# appstore-sb3-test
This .sb3 is to be ran in Turbowarp. It is a demo of the appstore for NitroOS Alpha 1.2.2.
It uses Iframe to run embeddable games which link's are converted into a code and not a long link.
Instructions to use are inside the project.
